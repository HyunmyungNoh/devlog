# 자바스크립트 기본
웹 브라우저에서 동작하는 유일한 프로그래밍 인터프리터 언어. 

명령형, 함수형, 프로토타입 기반 객체지향 프로그래밍 지원

모든 브라우저는 자바스크립트를 해석하고 실행할 수 있는 자바스크립트 엔진 내장

(브라우저에서 지원하지 않는 최신 기능 사용 시 Babel과 같은 트랜스파일러를 사용해 고사양구현 소스를 저사양으로 다운그레이드 진행)
